## 1. 개요
본 보고서는 사용자가 제공한 NABIS (국가균형발전 종합정보시스템) 통계정보 상세조회 링크(`https://www.nabis.go.kr/totalStatisticsDetailView.do?menucd=168&menuFlag=Y`)의 화면 구성을 파악하고, 주요 지표인 '핵심·객관지표'와 '주관지표' 간의 페이지 이동 및 탐색이 원활하게 작동하는지 점검한 결과를 정리한 문서입니다.

## 2. 페이지 화면 구성
해당 페이지(균형발전지표)는 사용자가 통계 데이터를 직관적으로 확인하고 지역별로 탐색할 수 있도록 다음과 같은 주요 UI 요소들로 구성되어 있습니다.

*   **상단 탭 (지표 분류)**: 
    화면 중앙 상단에 **[핵심·객관지표]**와 **[주관지표]**라는 두 개의 큰 탭 버튼이 가로로 나란히 배치되어 있어, 사용자가 원하는 지표 유형을 쉽게 선택할 수 있습니다.
*   **좌측 메뉴 (지역 계층구조 트리)**: 
    화면 좌측 하단 영역에는 **전국 ➔ 시도 ➔ 시군구**의 계층구조를 가진 지역 선택 메뉴가 트리(Tree) 형태로 제공됩니다.
    *   최상위 노드인 `전국` 아래에 `서울특별시`, `부산광역시`, `경기도` 등의 시·도가 나열됩니다.
    *   세부 구조가 존재하는 시·도 또는 시·군·구 항목은 확장/축소하여 하위 행정구역을 선택할 수 있으며, 이를 통해 원하는 특정 지역의 데이터만 우측 화면에 불러올 수 있습니다.
*   **우측 및 하단 (데이터 조회 영역)**: 
    선택된 지표 탭(객관/주관)과 연도(예: 2019년 ~ 2022년), 그리고 좌측 트리에서 선택한 특정 지역에 대한 상세 지표 값(종합, 안전, 환경 등 부문별 수치 및 전국 평균과의 비교)을 테이블 형태로 제공합니다.

## 3. 지표 간 페이지 이동 및 탐색 점검
사용자의 주요 관심사인 '핵심·객관지표'와 '주관지표' 카테고리 간의 페이지 이동 동작을 테스트한 결과는 다음과 같습니다.

*   **동적 화면 갱신 (Dynamic Content Update)**:
    *   기본적으로 랜딩 시 **'핵심·객관지표'**가 선택되어 있습니다. 
    *   옆의 **'주관지표'** 탭을 클릭하면 브라우저의 전체 페이지가 새로고침되거나 다른 URL 주소로 이동하는 것이 아니라, 기존 페이지(`totalStatisticsDetailView.do?...`) 내에서 우측의 데이터 명칭과 수치 테이블 콘텐츠만 주관지표 내용(예: 삶의 만족, 생활 만족 부문 등)으로 동적으로 빠르게 변경됩니다.
*   **원활한 상호 전환 지원**: 
    *   '주관지표'를 보고 있는 상태에서 다시 **'핵심·객관지표'** 탭을 클릭하면 즉각적으로 원래의 객관지표 데이터(예: 인구증감률, 등)로 정상 전환됩니다. 
    *   이 전환 시스템은 탭 형식의 UI를 통해 구현되어 있어 매우 빠르고 쉽게 두 지표 간의 분석 화면 이동이 가능합니다.

## 4. 결론
NABIS 균형발전지표 페이지는 사용자가 요구한 **'전국 - 시도 - 시군구' 단위의 계층적 지역 탐색 기능**을 좌측 트리를 통해 완벽히 지원합니다. 또한 **'핵심·객관지표'와 '주관지표' 간의 전환은 별도의 페이지 리디렉션 없이 탭(Tab) 클릭 이벤트만으로 동일 화면 안에서 안정적이고 직관적으로 구동**되어 매우 효율적인 탐색이 가능합니다.
