## 1. 전체 레이아웃 구조
화면(Page 1, 해상도 1512x982)은 크게 좌측의 **사이드바(Sidebar)**와 우측의 **콘텐츠(Contents)** 영역으로 구성되어 있습니다.
- **사이드바 영역**: `width: 200px`, 화면 좌측(`x: 0`)에 고정
- **콘텐츠 영역**: `width: 1312px`, 화면 우측(`x: 200`)에 배치되어 메인 데이터 시각화를 담당

## 2. 사이드바 (Sidebar) 요소
컨트롤 패널 역할을 하며 사용자 입력 및 요약 정보를 보여주는 계층형 구조입니다.

| 구성 항목 | 타입 | 설명 및 구현 요구 사항 |
| :--- | :--- | :--- |
| **타이틀** | `Text` | 상단에 "균형발전상황판" 이라는 메인 제목 위치 |
| **기준년도** | `Select Field` | 조회를 희망하는 기준 년도 선택 (2021년~2025년 데이터 동적 바인딩) |
| **관심지표** | `Select Field` | 사용자가 탐색하고자 하는 주요 지표 항목을 선택하는 드롭다운 필드 |
| **지자체별 지표 요약** | `Textarea` | 라벨 `시도명 - 시군구명`. 선택된 지역의 2021년~2025년 변화량 및 해당 시도의 지표값, 전국 단위 지표값과의 비교 수치 제공 |
| **푸터** | `Text` | 하단에 "Developed by Q" 정보 표출 |

## 3. 결과 콘텐츠 (Contents) 요소
인터랙티브 맵이 표시되는 우측 메인 영역입니다.

- **인터랙티브 지도 위치**: `contents` 영역 전체를 활용하여 인터랙티브 지도를 표출합니다. 사이드바에서 설정한 조건(기준년도, 관심지표)에 맞추어 지도의 폴리곤 색상 및 데이터 툴팁이 렌더링되도록 연동되어야 합니다.
